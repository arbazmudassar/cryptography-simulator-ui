# 🔐 Cryptography Simulator UI

An interactive front-end interface that allows users to simulate classical cryptographic algorithms. This project visually represents encryption and decryption processes, providing a clear and engaging learning experience.

---

## 📌 Overview

Understanding cryptography can be challenging. This UI project simplifies the concepts by providing a visual simulation of how text is transformed using different algorithms. The project emphasizes clean design, responsiveness, and usability, suitable for educational purposes or portfolio display.

---

## 🚀 Features

* 📝 Input fields for text and encryption keys
* 🔑 Algorithm selection (e.g., Caesar Cipher, Rail Fence Cipher)
* 🔄 Step-by-step visual representation of encryption and decryption
* 🖥️ Responsive layout for desktop, tablet, and mobile
* 🎨 Clean and structured design using Bootstrap

---

## 🛠️ Tech Stack

* **HTML** – Page structure
* **CSS** – Styling and layout
* **Bootstrap** – Responsive design
* **JavaScript** – Algorithm simulation and visualization
